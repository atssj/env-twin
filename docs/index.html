🔧 env-twin

Seamlessly Synchronize Your Node.js Environment Configurations

The ultimate CLI tool to keep your environment variable configurations perfectly synchronized across all .env* files

Features

  • ✅ Automatically sync environment variable keys across multiple .env* files
  • 🔐 Secure sensitive data by sanitizing values in .env.example files
  • 🔄 Preserve comments, formatting, and file structure
  • 💾 Smart backup and rollback management
  • 🌏 Cross-platform support: Windows, macOS, and Linux
  • 📦 Zero external dependencies

Quick Install

npm install --save-dev env-twin
